# Example Workflow

Here's a complete workflow for scanning a Foundry project.

## 1. Install Sunya

**Linux & macOS:**

```bash
curl -sSL https://raw.githubusercontent.com/13x54n/sunya-landing-page/main/package/bin/install | sh
```

**Windows (PowerShell):**

```powershell
irm https://raw.githubusercontent.com/13x54n/sunya-landing-page/main/package/bin/install.ps1 | iex
```

## 2. Initialize Foundry project

```bash
forge init my-token
cd my-token
```

## 3. Create config

Create `sunya.config.json` in the project root (Foundry uses `./src`):

```json
{
  "contractsDir": "./src",
  "output": { "format": "table" }
}
```

## 4. Run scan

```bash
sunya scan
```
